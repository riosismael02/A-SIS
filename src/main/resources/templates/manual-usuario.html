Manual de Usuario

Bienvenido al Manual de Usuario de A-SIS. Aquí encontrarás toda la información necesaria para utilizar las funcionalidades del sistema de asistencia, desde la carga de planillas hasta la gestión de ausencias.

Detalle de asistencia individual

Esta opción es ideal para consultar la asistencia de un empleado específico en cualquier rango de fechas que necesites, sin límite de tiempo.

Listar planillas mensuales

En esta sección puedes ver las últimas 12 planillas de asistencia cargadas en el sistema. Cada planilla muestra un resumen con el detalle de horas trabajadas por tipo para cada empleado, información clave para calcular sueldos.

Cargar planilla mensual

El día 21 de cada mes deberás cargar la planilla con los registros de asistencia extraídos del reloj biométrico. Sigue estos pasos:

  1. Conecta un pendrive al reloj biométrico.
  2. En el reloj, selecciona: USB > Extraer todo tiempo de conexión.
  3. Una vez copiados los datos al pendrive, retíralo de forma segura.
  4. Conecta el pendrive a una computadora con acceso al sistema A-SIS.
  5. Busca el archivo ALOG_001.txt, haz clic derecho y elige Abrir con... > Excel.
  6. En Excel, guarda el archivo como Libro de Excel (*.xlsx).
  7. Sube este archivo guardado desde tu computadora al sistema A-SIS usando la opción "Guardar y procesar".
Tras cargar y procesar el archivo, podrás ver la planilla mensual con el resumen de asistencia.
Cargar empleado

IMPORTANTE: Antes de dar de alta un empleado, verifica en "Gestionar Empleados" que no esté registrado para evitar duplicados.

Si el empleado no existe, completa sus datos en el formulario correspondiente.

Nota: El segundo horario normal se debe cargar solo para ciertos empleados, como los del área de tránsito o del Hogar de Adultos Mayores.

La flexibilidad horaria es igual para todo el personal.

Gestionar Empleados

Esta función te permite corregir datos o confirmar si un empleado ya está registrado.

Usa esta opción con precaución, ya que modificar datos incorrectamente puede generar errores en las planillas mensuales.
Justificar ausencias

Utiliza esta opción únicamente cuando un empleado se ausenta justificadamente.

También sirve para registrar situaciones en las que el empleado llega tarde o se retira antes de tiempo, siempre que sea con justificación.

Ver y asignar feriados

Los feriados deben estar cargados antes de poner en marcha el sistema A-SIS para que el cálculo de horas sea correcto.

Si detectas errores en algún feriado, puedes eliminarlo y cargar la información correcta.

Para más información detallada contacta al administrador.
